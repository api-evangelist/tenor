# Tenor (tenor)

Tenor is Google's GIF search engine REST API that enables developers to integrate animated GIF search, trending GIFs, featured content, and category browsing into applications. The API processes over 300 million searches per day across 45+ languages with localized, region-appropriate content. Note: As of January 2026, Google is no longer accepting new API clients; the API is scheduled for sunset on June 30, 2026.

## APIs

### Tenor GIF Search API

REST API for searching and browsing animated GIFs and stickers. Provides endpoints for GIF search, trending content, featured GIFs, categories, search suggestions, autocomplete, and share registration. Built on Google's Tenor platform with support for 45+ languages and localized content. Base URL: https://tenor.googleapis.com/v2/

- **Human URL:** [https://developers.google.com/tenor](https://developers.google.com/tenor)
- **Base URL:** `https://tenor.googleapis.com/v2`
